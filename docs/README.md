# Path Mapper Documentation

Path Mapper is a lightweight, playback-only Virtual Tabletop (VTT) for Pathfinder 2e and other square-grid TTRPGs. All content --- adventures, groups, and maps --- is authored externally using text editors and GIMP, then packaged into ZIP files that the server loads.

## Reading Order

If you are new to Path Mapper, read these documents in order:

1. [Quick Start](quick-start.md) --- create your first adventure and group, build ZIPs, and run a session
2. [Groups](groups.md) --- complete reference for creating player groups
3. [Maps](maps.md) --- map construction with GIMP and the ORA layer naming convention
4. [Adventures](adventures.md) --- complete reference for creating adventures
5. [GM Guide](gm-guide.md) --- running a game session (includes "What your players see")
